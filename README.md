# Windows-Endpoint-Security-Monitoring

This project highlights a controlled Windows 11 virtual machine experiment designed to simulate and analyze core authentication events using Windows Event Viewer. By monitoring specific Security Event IDs, the project demonstrates how a successful login (Event ID 4624) establishes a baseline of normal user activity, while consecutive failed login attempts (Event ID 4625) and subsequent account lockouts (Event ID 4740) serve as crucial indicators of an active brute-force attack. Additionally, tracking unauthorized user creation (Event ID 4720) helps identify potential hacker persistence and privilege escalation mechanisms. Ultimately, the report concludes that proper local security auditing configurations are vital for providing the visibility and log correlation that SOC analysts need to detect threats and defend endpoints effectively.
